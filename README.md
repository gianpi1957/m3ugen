m3ugen
======

Create a m3u playlist with all files (ogg, mp3, mpc) in given directory. Uses
eyeD3 and pyvorbis libs to read music length.

Usage
-----

    python m3ugen.py <dir1> <dir2> ... <dirn>
   
Requirements
------------

- eyeD3
- pyvorbis
